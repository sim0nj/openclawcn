推荐：Discourse 自建方案

Discourse 是成熟的开源论坛系统，支持 SSO、私有部署与高性能缓存。示例部署方式如下（请在域名与邮箱设置处替换为你的信息）：

# 简化版 docker-compose.yml（示意）
version: "3.9"
services:
  app:
    image: discourse/base:2.0.202
    container_name: discourse
    restart: unless-stopped
    environment:
      - DISCOURSE_HOSTNAME=forum.example.com
      - DISCOURSE_SMTP_ADDRESS=smtp.example.com
      - DISCOURSE_SMTP_USER=user@example.com
      - DISCOURSE_SMTP_PASSWORD=secret
    volumes:
      - ./data:/var/www/discourse
    ports:
      - "80:80"
      - "443:443"

生产环境建议使用官方 discourse_docker 或 Helm Chart，并接入对象存储与外部数据库以便扩展。

单点登录（SSO）

  • 企业 SSO：OAuth2/OIDC（支持 Azure AD、GitHub、GitLab、Gitee 等）。
  • 站点级 SSO：可将中文站登录与论坛绑定，实现一键登录。

发帖规范

  • 标题简明扼要，包含模块/版本/关键症状。
  • 正文提供环境、配置片段、错误日志及最小可复现步骤。
  • 如为功能建议，请注明业务背景与预期收益；欢迎附上草图或 API 提案。
